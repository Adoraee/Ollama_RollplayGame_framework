# Ollama 角色演绎模板 (Roleplay Framework)

>简单的本地跑团rp模板，Qwen、Llama 3 等大模型已内置相关跑团规则，具体的prompt可以自行搜索添加。

## 使用方法

### 第一步：下载模板
下载本项目中的 `Modelfile-Template` 文件。

### 第二步：在模板中填写角色
用任意文本编辑器打开该文件，将其中的 `[填空]` 部分替换为你构思的设定。

完成后，将文件名重命名为 `Modelfile`（注意：不需要任何后缀名）。

### 第三步：在本地运行
打开你的终端（命令行），进入你存放该 `Modelfile` 的文件夹，运行以下两条指令：
  1. **创建角色模型**
   ```bash
   ollama create my-character -f ./Modelfile
   ```
  2. **开始对话**
   ```bash
   ollama run my-character
   ```

## 如果需要DM/KP的话
如果你让大模型AI记玩家的血量（HP）、法术位或者背包里的子弹数，不出五个回合，它就会开始胡言乱语（比如给你凭空变出一把加特林，或者死掉的 NPC 突然复活）。

为了防止它遗忘玩家的数值，需要要求它在每次回复的末尾，强制打印一遍当前状态。

```markdown
你的每次回复，必须在最末尾附带以下 Markdown 状态栏：
---
**当前状态记录：**
- 玩家 HP：[数值]
- 玩家 理智值 (SAN)：[数值]
- 当前场景：[一句话总结位置]
